# Escape From Yerba Buena

*Escape From Yerba Buena* is a turn-based 1850s western roguelike built with [Bevy](https://bevyengine.org/) and [bevy_ratatui](https://github.com/ratatui/bevy_ratatui). The current game text in the code frames you as a prisoner blasted out of a riverfront cell while four gangs tear Yerba Buena apart, and your win condition is simple: reach any edge of the city alive before the whole place turns into a killbox.

The game leans into frontier chaos rather than a single scripted path. You move through a procedurally generated western town, loot what you can, manage ammo and stamina, and survive long enough to slip out while rival factions fight each other in the streets. The in-game help also calls out key tactics directly: stay moving, pick your fights, take what you need, and get out.

```bash
cd roguelike
cargo run --release
```
